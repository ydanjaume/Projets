> WAFE est un univers fantastique peuplé d'animaux, d'humanoïdes et d'éléments vivants. Le jeu principal est un **turn-based RPG sur tuiles** inspiré du mode des échecs (chaque joueur est un roi entouré de pions). Un **arbre créateur** distribue les âmes comme des graines — il récupère les morts par ses racines. Le meilleur joueur de chaque arme devient le **maître** de cette arme.

---

## Concept de jeu

- Turn-based RPG sur tuiles, mode Duelyst / échecs
- Sur le plateau : le personnage principal est le **roi d'échec**, il a des pions
- La première attaque humaine / animale / élément = coup de base
- 7 slots d'attaque, dont le **premier obligatoire** correspondant au type de l'être vivant
