# Sources et restitutions historiques

Ce répertoire regroupe tout ce qui est à l’origine de la bibliothèque active [`Atelier`](../Atelier/README.md).

## Sources brutes

- `Origine/Github/` — document du projet GitHub Sheepering ;
- `Origine/Notion/` — exports Markdown, CSV et images de Notion ;
- `Origine/OneNote/` — fichiers `.one`, exports `.docx` et index du bloc-notes.

## Restitutions IA

- `Restitutions/ChatGPT/` — versions V1 et V2 produites par ChatGPT ;
- `Restitutions/Claude/` — versions V1 et V2 produites par Claude ;
- `Restitutions/ChatGPT/Outils/` — deux scripts conservés uniquement pour reproduire les restitutions ChatGPT.

Les restitutions sont des archives de travail dérivées des sources. Le contenu consolidé à maintenir se trouve désormais dans `Atelier/`.
