# WAFE - Donjon

Module donjon à base de cartes pour le système WAFE.

Inspiré de *Voice of Cards: The Isle Dragon Roars*.

---

## Sections OneNote d'origine

Le fichier `WAFE - Donjon.one` contenait :
- **JEU V1** — première version du module
- **JEU V2** — version révisée
- **Personnage** — fiche de personnage pour le mode donjon
- **Ennemis** — bestiaire
- **Général** — notes générales

---

## Système de jeu

Tout se joue avec des cartes : plateau, équipement, compétences, ennemis.

Structure de carte de personnage observée :
- Cartes (deck de compétences/équipement)
- Techniques
- Talents
- Équipement
- Équipe / Formation
- Niveau / Prochain niveau
- Données (stats)
