# YNSECTS

Projet de jeu thématique insectes.

---

## Concept

Jeu mettant en scène des insectes comme personnages / factions.

**Inspirations visuelles notées :**
- *Hollow Knight* — style artistique d'insectes anthropomorphiques
- *Final Fantasy XIV* : jobs adaptés en insectes (concept fanart)

---

## Sections OneNote d'origine

Le fichier `YNSECTS.one` contenait :
- **Général** — concept et règles
- **Fanart** — références visuelles et inspirations artistiques
