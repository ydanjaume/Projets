## Concept

Recréer la tension d'un match de volley inspiré de l'univers *Haikyuu* : chaque point se joue action par action, chaque poste a un rôle précis, et les joueurs stars ont des capacités spéciales qui peuvent renverser un rally.

Le plateau représente les **deux camps séparés par le filet**, avec des zones de terrain (fond de court, zones d'attaque, filet).

---

## Structure d'un rally

Un point se joue en séquence :

```
Service → Réception → Passe → Attaque → (Contre / Défense) → Point
```

Chaque étape est résolue par un **jet de dé modifié par le poste du joueur actif** et les cartes jouées.

---

## Postes — Asymétrie des joueurs

| Poste | Rôle dans le rally | Bonus |
|---|---|---|
| **Passeur** | Transforme la réception en passe parfaite | +dés sur la passe, peut orienter l'attaque |
| **Attaquant principal** | Frappe décisive | Haute attaque, peut tenter le smash |
| **Libéro** | Spécialiste de la réception | Relance les balles impossibles, immunisé aux smash directs |
| **Serveur** | Lance le rally | Peut forcer des services flottants ou en puissance |
| **Contre** | Blocage au filet | Peut annuler une attaque adverse |
| **Remplaçant** | Carte tactique | Peut entrer en jeu pour un rally clé |

---

## Mécaniques principales

### Cartes tactiques
Chaque joueur a une main de cartes représentant des **tactiques d'équipe ou actions spéciales** :
- *Attaque en pipe* — attaque surprise depuis le fond
- *Bloc en mur* — deux joueurs contrent ensemble
- *Service en jump* — service difficile à recevoir
- *Synchronisation* — combo passeur + attaquant, +2 dés

### Jauges de momentum
Comme dans Haikyuu, le **momentum** change le cours du match :
- Gagner 3 points d'affilée → **momentum** → bonus de dés pour l'équipe
- L'adversaire peut "casser" le momentum avec une action défensive réussie

### Le set et le match
- Un set = premier à **25 points** (avec 2 points d'écart)
- Un match = meilleur de 3 ou 5 sets
- La rotation des joueurs sur le terrain est obligatoire à chaque point gagné
