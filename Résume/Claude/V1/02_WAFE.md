# WAFE — Univers complet
> Sources : OneNote `WAFE.docx`, `WAFE - Banalités.docx`, `WAFE - King battle.docx`, `WAFE - LandProtect.docx`

WAFE est un projet d'univers complet (RPG / jeu de stratégie) avec plusieurs modes de jeu. Les notes couvrent la période novembre 2020 à décembre 2021.

---

## WAFE — Général (WAFE.docx)
*(3 décembre 2021)*

- Le meilleur joueur de chaque arme apparaît comme le maître de cette arme.
- Le jeu serait un **turn-based RPG sur tuiles** au mode échec, en mode Duelyst.
- Le créateur de la planète serait un **arbre qui distribue les âmes** comme des graines, ce qui donne la vie aux êtres vivants. Il récupère les âmes mortes avec ses racines pour se nourrir.
- Sur le plateau le personnage serait le **roi d'échec** et il aurait des pions.
- La première attaque humaine/animale/élément est un coup de base.

**Références :**
- Skyrim : arbre de talents
- New World : elements
- Albion : arbre d'apprentissage
- Child of light : tours
- Pokemon : attaque/Types
- Danmachi : talents, tour, dieux
- Chained Echoes : spells
- Seven deadly sins => pouvoirs dieux

## LST — Armes *(14 novembre 2021)*

**Fonctionnement :**
Les doubles armes et deux mains sont une amélioration de l'arme simple.
*Exemple : Épée simple → Dual sword → Double hand sword*

**Liste des armes :**
Arme Pugilat, Lance/Hast, Éventail (?), Arc, Arbalète, Épée, Rapière, Espadon, Hache, Marteau, Masse, Bouclier, Dagues, Bâton, Arme de jet, Torche, Bolas, Faux, Trident, Boomerang, Armes à feux

  - Dagues : Simple, Dual, DoubleHand

- **Baton**
  - Lance : Simple, Dual, DoubleHand, Hast, Trident

## WAFE — King Battle *(WAFE - King battle.docx)*
*(19 novembre 2020)*

**Concept :** Armello-like

- Système de **décrets par jour** choisi par le héros le plus *mighty*
- Systèmes de **quêtes** en fonction de certaines cases du tableau
- Dés avec : Attaque (1, 2, 3) / Défense (1, 2, 3)
- Dés avec : Chaque élément + ombre/lumière

**Séquence de combat :**
1. (1) Magie : nombre de dés correspondant à nos sorts
2. (Pour l'initiateur) Attaque : nombre de dés d'attaque
3. (Pour le receveur) Défense : nombre de dés de défense
4. (Pour le receveur) Attaque : nombre de dés d'attaque
5. (Pour l'initiateur) Défense : nombre de dés de défense

**Les donjons :** lancer des dés qui correspondent au donjon (EAU / TERRE)

**Légende des éléments sur la carte :**

| Code | Signification |
|------|--------------|
| E | Eau |
| A | Air |
| F | Feu |
| T | Terre |
| ET | Marécage |
| AE | Glace |
| FA | Foudre |
| TF | Lave |
| X | Palais |
| D | Donjon |
